# Frontend

## Project setup
```
npm install
```

### Copies necessary files from web-ifc installed dependencies
```
npm run preserve
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
